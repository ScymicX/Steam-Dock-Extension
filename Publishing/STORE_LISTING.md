# Microsoft Store listing

## Product name

Steam Dock Extension

## Short description

Search installed Steam games and see online friends in the PowerToys Command Palette Dock.

## Description

Steam Dock Extension brings your local Steam library and online friends to the
PowerToys Command Palette Dock.

Browse installed games alphabetically, search by title, view local cover artwork
and recent-play information, and launch games directly through Steam. Add the
online friends band to see who is available or currently playing, then select a
friend to open a Steam chat.

The game library is discovered from local Steam files. Online friends are
optional and require your own Steam Web API key and a public Steam friends list.
The API key is stored in Windows Credential Manager.

This is an independent project and is not affiliated with or endorsed by Valve
Corporation or Microsoft.

## Features

- Search and launch locally installed Steam games
- Local Steam cover artwork and game details
- Online and in-game friends in the Command Palette Dock
- Open Steam chat directly from a friend
- Configurable Steam path, account, refresh interval, and friend limit
- No developer-owned server, analytics, or telemetry
